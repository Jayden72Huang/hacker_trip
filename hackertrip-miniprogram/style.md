# HackerTrip 小程序 · 视觉规范 style.md

> 排版基底 = 方案 A「舒适留白」；模块结构 = 方案 B「结构化卡片」。颜色主题待单独确定（当前暂用深色玻璃过渡色）。
> 单位：小程序 rpx（750rpx = 屏宽）。8rpx 栅格。

## 字阶 Typography
| 角色 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| Display/H1 | 52rpx | 800 | 1.34 | 页面主标题 / Hero |
| H2 / Section | 36rpx | 700 | 1.4 | 区块标题 |
| Card Title | 34rpx | 700 | 1.42 | 卡片标题 |
| Body | 28rpx | 400 | 1.65 | 正文 |
| Secondary | 26rpx | 400 | 1.6 | 次要说明 |
| Caption | 24rpx | 400 | 1.5 | 元信息（日期/城市） |
| Micro / Chip | 23rpx | 500 | 1.3 | 标签 |
- 字族：-apple-system / PingFang SC。字距：标题 +0.3~0.5rpx，正文 +0.2rpx。

## 间距刻度 Spacing（8rpx 栅格，舒展取向）
| token | 值 | 用途 |
|-------|----|------|
| xs | 8rpx | 紧邻元素 |
| sm | 16rpx | 标签间距 |
| md | 24rpx | 元素组内 |
| lg | 32rpx | 卡内分区 |
| xl | 40rpx | 页边距 / 大间隔 |
| 2xl | 48rpx | 区块之间 |
- 页面横向 padding：**40rpx**
- 卡片内边距：**36rpx**
- 卡片之间间距：**28rpx**
- 区块（section）之间：**48rpx**
- 卡内节奏：标题 →(20rpx) 元信息 →(20rpx) 赛道标签 →(28rpx + 细线) 底栏

## 圆角 Radius
| 元素 | 值 |
|------|----|
| 卡片 | 30rpx |
| 输入框 / banner | 24rpx |
| chip / 按钮 | 999rpx（全圆） |
| 小标签块 | 12rpx |

## 阴影 Elevation
- 卡片：`0 16rpx 48rpx rgba(0,0,0,.4)`（从深色背景浮起）
- 主按钮：`0 14rpx 40rpx rgba(accent,.4)`
- glow（强调）：`0 0 50rpx rgba(accent,.3)`

## 组件 Components（模块结构 = 方案 B）
- **黑客松卡片**：标题（34/700）→ 元信息行（📅日期 · 📍城市 · 形式高亮）→ 赛道 chips（最多 4）→ 细线分隔 → 底栏（🏆奖金 + 状态徽章）。内边距 36rpx。
- **精选大卡**：首张/进行中可加 kicker 小标题 +「FEATURED」，视觉权重更高。
- **分组**：列表按状态分组（🔥进行中 / 🗓即将开始 / 已结束），每组带小标题 + 「全部 ›」。
- **chip**：9rpx 22rpx，全圆；选中态用 accent 填充。
- **主按钮**：高 92rpx，全圆，accent 渐变 + 阴影；按压 scale .98。
- **输入框**：高 92rpx，圆角 24rpx，左 icon + placeholder。
- **空状态**：居中 emoji + 一句说明 + 主按钮。
