## 已沉淀的硬规则（每页必守）

1. **禁 emoji**：跨端渲染不稳（会变缺字形造成幽灵缩进）。图标用矢量形状；TabBar 母版组件（193:50）已内置矢量图标，直接实例化后 detach 改选中态。
2. **页面高度 = 完整内容高度**（可滚动），不压缩间距塞一屏。区块间距 24-32 / 卡间距 12 / 页边距 16。
3. **投影分级定主次**：每页只有一个焦点元素用 lg(6)（身份卡等可导出资产用 xl 8）；标准卡 md(4)；列表/小件 sm(2)；"安静行"集合（如 Top5 的 #2-5）可去投影只留黑边。
4. **黄色预算 ≤5%**：只用于小角标/当前阶段段/热度徽章/CTA 圆钮。选中 chip = 蓝底。
5. **标题模式**：tab 页 = 大标题 30 Black 左对齐；焦点流 = ← + 17 Black 导航行（06 身份卡这类"我的域子页"可升 26）。
6. **标准页面结构（2026-06-11 定稿，所有 Hi-Fi 页必须用）**：root 统一 **393×852**（小程序视口）+ `overflowDirection="VERTICAL"` + `clipsContent`；子节点顺序 = `__content`（滚动内容，padding 16/底部 110 防底栏遮挡）→ `__topbar(fixed)`（状态栏，焦点流页含 ← 导航行，canvas 底色防透出）→ 底栏（tabbar / 命令栏 / 详情页双 CTA）；`numberOfFixedChildren` = 固定子节点数（顶栏+底栏）。**禁止再出现随内容变高的页面 frame**——完整内容长度由 `__content` 承载，视口恒定。
7. **Figma API 坑**：纵向容器定宽用 `counterAxisSizingMode=FIXED`（horizontal 行才用 primaryAxis）——搞反会把高度冻结裁掉子元素；改布局轴后要重设 sizing；克隆+改字号前必须预载该字体（含 Space Grotesk），否则整个调用回滚；长文本必须 `textAutoResize="HEIGHT"` + 固定宽度才会换行。
