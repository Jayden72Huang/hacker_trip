## Why This Direction

Hackathons are competitions with urgency, countdowns, rankings, and event heat. Infrared Arena makes those states visually obvious while keeping the interface product-like.

## Product Logic

- Red is used for active competition state, countdown, heat, and CTA.
- Black event covers create live-arena impact.
- Yellow is reserved for winner/award moments.
- White surfaces keep utility clear despite the strong accent color.

## Fit For HackerTrip

Best for users already in competition mode: registered, building, submitting, or preparing for demo day. It makes progress and deadlines impossible to miss.

## Risk

Red can feel stressful if overused. It is less suitable for calm onboarding and long browsing sessions.

## Recommendation

Use this if the product wants to emphasize action, urgency, and live event operations.
