## Why This Direction

Carbon's logic is strong neutral layering, clear tokens, and sparse purpose-driven color. This maps well to HackerTrip as a "hackathon OS": users need status, progress, tasks, project metadata, and technical confidence.

## Product Logic

- Black and graphite create a command-center feeling.
- Terminal green is used for active matching, progress, and successful states.
- Orange stays semantic for deadlines and heat, not decoration.
- Cards and lists remain structured so event data can scale.

## Fit For HackerTrip

Best for advanced builders, AI agents, GitHub-heavy users, and "workspace" positioning. It makes HackerTrip feel like an operational tool for serious participants.

## Risk

It is colder and more technical. New users may feel it is less approachable than the pastel or cobalt directions.

## Recommendation

Strong candidate if HackerTrip wants a distinctive developer-first identity.
