## Why This Direction

Material 3 encourages expressive but systematic use of color. Space Violet Research turns HackerTrip into an AI research and exploration cockpit, which matches the product's AI matching, event discovery, and agent configuration ambitions.

## Product Logic

- Violet is used as the main action and matching color.
- Deep navy gives event posters and status panels more depth.
- Amber is reserved for countdown and urgent actions.
- The space/research motif supports "discover unknown opportunities" without becoming a normal event listing app.

## Fit For HackerTrip

Best for an AI-forward product identity. It visually supports future modules such as Agent configs, skills library, matching scores, and AI recommendations.

## Risk

Purple can easily become generic AI SaaS. The system must keep white surfaces and restrained use of violet to avoid visual fatigue.

## Recommendation

Recommended as one of the top two options if HackerTrip wants a more ownable AI/research identity than standard blue.
