微信小程序规范要点

导航custom，避让右上 88px 胶囊
触控区≥ 44×44 pt（HIG/微信）
底部安全区env(safe-area-inset-bottom)
抽屉/弹层page-container（非真实跳转）
页面跳转navigateTo / switchTab
消息推送requestSubscribeMessage
登录头像昵称填写 + 手机号快捷
