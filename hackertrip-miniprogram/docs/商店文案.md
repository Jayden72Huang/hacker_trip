# HackerTrip 小程序 · 商店文案

## 名称
HackerTrip 黑客松

## 简介（一句话，≤ 32 字）
发现黑客松，生成你的选手身份卡，找到队友。

## 详细描述（备用，用于推广/分享）

HackerTrip 是黑客松选手的移动端总入口：

🔍 **发现黑客松** —— 汇聚全球优质黑客松，按城市、赛道、线上线下筛选，一眼看到奖金池与报名截止。

🎴 **选手身份卡** —— 填几个技术栈，AI 帮你判定「你是哪种黑客松角色」：从零到一搭建者、炼丹师、像素雕花匠……生成炫酷身份卡，分享到群里找队友、找联合创始人。

🛰️ **Skills 同步** —— 在电脑上用 `/ht-scan-project` 扫描你的项目，一个配对码就把「Top5 黑客松匹配结果」同步到手机，随时查看、随时报名。

📌 **收藏与报名清单** —— 心仪的比赛一键收藏，报名清单帮你跟进 deadline。

100% 本地扫描、代码不上传。来 HackerTrip，让每一行代码都找到它的舞台。
