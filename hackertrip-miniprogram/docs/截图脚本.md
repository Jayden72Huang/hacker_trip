# HackerTrip 小程序 · 截图脚本

> 用于商店展示的 5 张截图。开发者工具模拟器（iPhone X / 375×812）或真机截图均可。
> 截图前在「发现」页确保有数据、在「同步」页点过「载入演示数据」。

| # | 页面 | 操作 | 看点 |
|---|------|------|------|
| 1 | 发现 | 进入首页，滚动到能看到 Hero + 搜索 + Skill banner + 2 张黑客松卡 | 主功能一目了然 |
| 2 | 详情 | 点开第一个黑客松 | 赛道/奖金/报名按钮 |
| 3 | 卡片 | Tab 切到「卡片」，等 Canvas 渲染出身份卡 | 核心亮点：身份卡视觉 |
| 4 | 同步 | Tab 切到「同步」，已载入演示数据状态 | Skills 同步差异化卖点 |
| 5 | 结果 | 从同步页点「查看匹配结果」 | Top5 匹配分数条 |

## 加工建议
- 顶部可叠加一句卖点文案（如截图 3 配「填几个技术栈，AI 帮你判定角色」）
- 保持深色背景，紫色 accent，风格统一
- 可用项目内 guizang-social-card / nano-banana-pro skill 做带文案的展示图
