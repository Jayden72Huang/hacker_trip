## 一、微信公众平台后台（https://mp.weixin.qq.com）

用 `wxcd9b63c...` 小程序账号登录，依次配置：

### 1. web-view 业务域名 🔴 提审必须
位置：**开发管理 → 开发设置 → 业务域名**
作用：详情页"去官方报名"用 web-view 加载赛事官网，不配真机打不开。
要添加的域名（下载校验文件放到对应服务器，或如果是第三方域名走业务域名校验流程）：
```
www.competehub.dev
www.adventurex.org
www.ithome.com
zhuanlan.zhihu.com
www.geekpark.net
news.sina.com.cn
hackathon.hrflag.com
```
> ⚠️ 业务域名需要域名所有者放校验文件，第三方赛事官网可能配不了。**替代方案**：详情页"去官方报名"改成 `wx.setClipboardData` 复制链接 + 提示"已复制，去浏览器打开"，或用 `wx.openOfficialAccountArticle`。如果配域名困难，告诉我，我改成复制链接方案（更稳，不依赖业务域名）。

### 2. 《用户隐私保护指引》🔴 提审必须
位置：**设置 → 服务内容声明 → 用户隐私保护指引**
作用：登录用了 `wx.getUserProfile` 收集头像昵称，不声明审核驳回。
操作：新增/更新指引，勾选收集的信息类型：**头像、昵称**（如果以后加位置/相册保存身份卡，也要勾"相册/存储"）。

### 3. 服务类目
位置：**设置 → 基本设置 → 服务类目**
建议：**工具 → 效率** 或 **教育 → 在线教育**（黑客松信息平台）。

### 4. 基本信息
位置：**设置 → 基本设置**
- 小程序名称、头像、简介、服务范围
- 确认主体信息（你已备案）
