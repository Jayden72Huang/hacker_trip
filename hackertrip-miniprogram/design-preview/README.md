# HackerTrip · Cyber-Mint 设计预览 (v2)

> 全新前端设计探索，**不影响** `miniprogram/` 现有代码。仅作设计验证用。

## 如何查看
直接用浏览器打开 `index.html`，4 个核心页面并排展示（手机尺寸）。

## 从 3 张参考图提取的设计语言
| 元素 | 提取结果 |
|------|---------|
| 背景 | 深空黑 `#050608` + 细点阵网格 + 十字星点 `+` 装饰 |
| 主强调色 | 薄荷青 `#4DE1FF` → 电光绿 `#5FFFC8` 荧光渐变 |
| 点缀色 | 紫 `#7C5DFF`（沿用项目原 accent） |
| 标志视觉 | Hero 点阵发光球（halftone glow orb） |
| 装饰符号 | 像素三角括号 `▸ ◂` 包裹小标题 |
| 数字/日期 | 等宽字体（monospace），青色，如 `06.24 – 08.02` |
| 卡片 | 玻璃质感 + 青色微光描边 |

## 4 个页面（核心功能完全保留，仅重做视觉）
1. **发现** — Hero 发光球 + 精选推荐横滑 + 城市筛选/搜索 + 黑客松卡片列表
2. **赛程** — 正在关注卡 + 等宽数字时间轴（7 段进度，呼应参考图3）+ 今日待办
3. **消息** — Haki AI 助手入口 + 分组消息（报名提醒/队友匹配/系统通知，含未读点）
4. **我的** — 参赛/举办身份切换 + 资料卡（头像+数据指标）+ 工具/设置列表
