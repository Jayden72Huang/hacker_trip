## 核心功能

### 1. 赛事发现 — AI 聚合，不再漏赛

自动爬取 DoraHacks、牛客、Devpost 等主流平台，覆盖国内外黑客松。搜索、筛选、收藏、报名提醒，一个入口搞定。

### 2. AI 项目匹配 — 终端里的赛事猎人

```bash
# 一行命令，AI 扫描你的项目，匹配最适合的黑客松
curl -sL hackertrip.space/install | sh
ht-scan
```

Agent 自动分析你的代码仓库（package.json、README、技术栈），从五个维度语义匹配最佳赛事：

> 主题契合 · 技术栈吻合 · 创新潜力 · 可行性 · 时间物流

100% 本地运行，代码不上传，中英双语支持。已适配 Claude Code / Cursor / Windsurf / Trae / Gemini CLI 等主流 AI 编程工具。

### 3. Haki — 你的 AI 黑客松导师

不是通用聊天机器人，是专为黑客松调教的 AI 导师：

- **赛题拆解**：上传赛题，分析赛道方向、评分权重、技术难度
- **创意风暴**：Socratic 式引导，帮你从模糊想法到可执行方案
- **项目规划**：自动生成任务看板，拆分、估时、分配一气呵成
- **Pitch 教练**：打磨演讲稿、演示脚本、准备 Q&A 应对策略

### 4. Hacker Bot — AI 驱动的团队协作

比赛期间的团队 AI 中枢：

- 实时协作（Web / Discord / Slack 多通道）
- 赛题分析报告一键生成
- 任务看板自动拆分与跟踪
- Artifacts 管理（可行性矩阵、时间轴、资源清单）

### 5. 项目名片 & 身份卡 — 让参赛成为资产

2 分钟录入，自动生成可传播的项目展示卡片。不是简历，是你的开发者战绩。

- **项目名片**：1200×630px，适配朋友圈/小红书/即刻一键分享
- **身份卡系统**：参赛履历时间线 + 技术栈配置 + 角色标识
- **可验证徽章**：「✓ 已核实获奖」，名片不只好看，更有公信力
- **个人主页**：`hackertrip.space/@你的ID`，你的黑客松 Portfolio
