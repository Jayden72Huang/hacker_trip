# 小红书发布文案

## 标题（20字以内）
一行命令，AI帮你匹配最适合的黑客松！

## 正文

开源了一个 AI Skill，在 Claude Code / Cursor / Windsurf 里输入一行命令，自动扫描你的项目代码，帮你找到最适合参加的黑客松比赛 🎯

---

💻 安装（10秒）：
```
curl -sfL hackertrip.space/install | bash
```

🎯 使用：
在任意项目目录输入 `/ht-scan-project`

---

✨ 它能做什么？

1️⃣ 自动扫描项目代码（package.json / requirements.txt / 源码 imports）
2️⃣ AI 语义分析你的项目做什么（不是关键词匹配！）
3️⃣ 从 20+ 场黑客松中匹配最适合的比赛
4️⃣ 给出匹配分数 + 推荐赛道 + Pitch 角度
5️⃣ 一键跳转报名页面

---

🔒 100% 本地运行，代码不会上传
🌏 支持中英文项目
⚡ 支持 6 款 AI 编程助手：Claude Code · Cursor · Windsurf · Codex · Trae · Gemini CLI

---

🤔 为什么做这个？

参加了 8 场黑客松之后发现，很多开发者不是没能力，是根本不知道自己的项目适合哪个比赛。

所以做了 HackerTrip（hackertrip.space）—— 一个黑客松聚合平台，把这个匹配能力做成了 AI Skill 开源出来，任何人都能用。
